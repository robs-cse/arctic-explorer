= ARCTIC EXPLORER =
(c) Robert Sison & Melody Wang, 2008

- Player interaction -
WSAD + arrow keys - movement
mouse movement - look
mouse click - fire signal flare

- Weather effects -
I - snow
O - fog

- Debug tools -
J - toggle fly mode
K - toggle lens flare
L - toggle time lapse
; - toggle clouds

- Ogre debug -
F - toggle render stats
R - cycle solid/wireframe/points rendering
T - cycle bilinear/trilinear/anisotropic filtering
